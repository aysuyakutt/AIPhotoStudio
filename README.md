# AIPhotoStudio

AIPhotoStudio is a modern Android application that lets users enhance, transform, and save their photos with AI-powered effects. The app combines a clean mobile experience with image editing workflows supported by generative AI and Firebase services.

## Features

- User authentication with Firebase
- Sign in with Google, email/password, or guest mode
- Apply predefined AI effects to selected photos
- Edit images with prompt-based AI instructions
- Save generated results to the user profile
- Download edited photos directly to the device gallery

## Tech Stack

- Android Studio
- Kotlin
- Jetpack Compose
- Firebase Authentication
- Firebase backend services
- Gemini-based AI integrations
- Git

## Development Notes

This project was one of my early Android development experiences. During the process, I learned and applied Jetpack Compose from scratch instead of building the interface with traditional XML layouts. I also worked on integrating AI services into a mobile app flow and managing the project structure with Git.

## Türkçe

AIPhotoStudio, kullanıcıların fotoğraflarına yapay zeka destekli efektler uygulayabildiği, düzenleyebildiği ve kaydedebildiği modern bir Android uygulamasıdır. Uygulama, mobil tarafta sade bir deneyimi yapay zeka destekli görsel düzenleme akışları ve Firebase servisleri ile birleştirir.
